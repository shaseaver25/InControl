Accessibility Rules (non-negotiable)

Touch targets

All interactive elements ≥ 56px tap target. No hover-only interactions. Spacing between targets ≥ 8px.

Contrast

All text ≥ 4.5:1 contrast ratio. Forest on Cream = 7.2:1. Amber on White = 3.1:1 (use for large text / icons only).

Motion

Respect prefers-reduced-motion. All animations have CSS fallbacks. No autoplay video with sound.

One thing at a time

One concept per screen. One question per interaction. Max 4 choices. No open text input as default.

Audio always available

Every text element has an audio narration equivalent. Captions on by default for all video.

No failure states

Errors are guidance, not punishment. No red ❌. Use amber 💡 hints before the user can fail. Never say "wrong."
